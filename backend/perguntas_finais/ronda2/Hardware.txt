O que significa GPU?	
Graphics Processing Units

O que é overclocking?
Overclocking é o processo de aumentar a velocidade de operação de um componente de hardware, para obter um desempenho melhor do que o especificado pelo fabricante.

O que é um socket de CPU?
Um socket de CPU é o local na placa-mãe onde a CPU é instalada
